# Phenolphthalein-indicator-acid-base-neutralization-titration
AI-driven phenolphthalein indicator acid-base neutralization titration for 2nd Intelligent Chemistry Challenge

**model.py**： 是模型文件  
**train.py**： 是调用模型训练的文件    
**predict.py**： 是调用模型进行预测的文件  
**class_indices.json**： 是训练数据集对应的标签文件   
**picture_capturing.py**： 是比赛和测试时使用的图片实时采集与检测程序 
**data** 文件夹将完成分类的照片分别放在“colorless”或“pink”子文件夹中即可
